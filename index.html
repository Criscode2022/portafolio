Acerca de mí

Acerca de mí

Filólogo de formación que se está reinventando en el ámbito laboral y personal al encontrar una gran pasión en la programación web.

Actualmente estoy formándome activamente a través de múltiples cursos online (incluido el Programa de Becas - Google Fundae) con gran vocación autodidacta y estoy cursando de forma presencial en Academia Postal A Coruña el Certificado de Profesionalidad de Nivel 3 "Desarrollo de aplicaciones con tecnologías web (IFCD0210)".

De amplia formación humanista, considero que mi experiencia anterior y mi vocación actual pueden dotarme de grandes aptitudes para el trabajo de programador con pasión por el trabajo en equipo. Además, desde esa formación humanística y artística tengo también mucho interés en el diseño UX/UI a fin de hacer de la navegación web una experiencia positiva que transmita emociones y los valores de las marcas en equidad para todos los usuarios. Para conseguir este objetivo, considero que la accesibilidad y la igualdad de oportunidades ocupa un lugar central en el desarrollo web, poniendo al usuario en el centro.

Destrezas y aptitudes

  • Especialización en el framework Angular con las librerías Angular Material y RxJS
  • Trabajo con API RESTful para gestionar peticiones HTTP GET, POST, PUT y DELETE (CRUD) con XMLHttpRequest, Axios, Fetch, HTTP Client (Angular) y HttpClient (Java)
  • Conocimiento avanzado de HTML, CSS (puro y con Bootstrap 5), Javascript y Typescript
  • Formación en UX, UI, accesibilidad y diseño adaptado a ordenadores de sobremesa, laptop, tablet, Ipad y smartphone
  • Back-end con Java: JDBC, Hibernate, Jackson y Jersey para crear microservicios RESTful que unan el Back-end con el Front-end por medio de JSON
  • Base de datos relacional MySQL con Workbench y phpMyAdmin
  • Control de versiones con Git y GitHub
  • Inglés intermedio: B2 certificado por Cambridge Assessment English (preparando C1)
  • Actitud activa y diligente para aprender y adaptarme
  • Trabajo en grupo con metodología Agile de Scrum
